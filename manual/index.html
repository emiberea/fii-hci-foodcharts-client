1. Introduction

The purpose of this application is to help users that are interested in nutritional values for food products, visualise data in an appealing way.

The application will be available in browser - desktop, tablet and phone.

FoodCharts (Visualious)

  • is a web application available for desktop and mobile devices
  • help the user select from a variety of filters
  • visualize data in multiple charts format
  • download the charts and save them locally on the computer
  • adjust font size
